# Library_Management_System-Mini_project
The Library Management System (LMS) is a software application designed to efficiently manage and automate the day-to-day operations of a library. It helps librarians and users handle book records, borrowing/returning processes, user registrations, and inventory management in a systematic and user-friendly way.
